# Tree_Cpp
트리 자료구조를 C++로 다뤄보는 레포지토리입니다.

## 1.이진탐색트리(재귀호출)
이진탐색트리는 평균적으로 O(log(n))의 시간 복잡도로 데이터의 삽입-검색-삭제를 할 수 있도록 한 트리입니다.  
이진탐색트리가 만족해야하는 조건을 재귀적으로 설명하면 다음과 같습니다.
```
1.이진탐색트리의 루트 노드의 키값은 좌측 자식 노드의 키값보다는 크고, 우측 자식의 키값보다는 작아야 한다.
2.왼쪽 자식의 하위 트리와, 우측 자식의 하위 트리도 이진탐색트리여야 한다.
```

### 탐색 과정
이진탐색트리에서 특정 키를 가진 노드를 찾아가는 과정은 다음과 같습니다.
```
1.자신이 찾고자 하는 키값과 루트 노드의 키값이 동일한지 확인한다. 동일할 경우 탐색 과정을 종료한다.
2.루트 노드의 키값보다 자신이 찾고자 하는 키의 값이 작다면 왼쪽 자식 노드드로 가고, 크다면 우측 자식 노드로 간다.
3.도착한 하위트리에서도 위 1번과 2번 과정을 반복한다.
```

### 깊이우선순회 과정 [1]
이진탐색트리에서의 깊이우선순회는 재귀를 사용하여 쉽게 구현할 수 있습니다.  
이때 순회를 하며 각 노드에 작업을 수행한다면, 그 작업의 수행 타이밍에 따라 '전위순회', '중위순회', '후위순회'로 나뉘게 됩니다.
```
depth_first_traverse(){
    preorder_traverse_task();           //전위순회로 수행할 작업
    left_child.depth_first_traverse();
    inorder_traverse_task();            //중위순회로 수행할 작업
    right_child.depth_first_traverse();
    postorder_traverse_task();          //후위순회로 수행할 작업
}
```
